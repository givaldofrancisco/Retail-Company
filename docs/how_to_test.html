How to Test the Retail AI Assistant

This guide mirrors the sections in AI Technical Assignment - Retail Company, listing commands and test suites that validate each requirement. Run the commands from the repository root ((repository root)) and activate .venv/bin/activate first.

The sections below link each assignment requirement to the CLI/Streamlit entry points, existing pytest files, or scripts that provide concrete evidence.

Setup & Common Routines

  • Environment: pip install -r requirements.txt then source .venv/bin/activate (or use Poetry).
  • Run CLI prototype: .venv/bin/python app.py --user-id manager_a --debug for interactive mode; add --input-file tests/manual_cli_inputs_en.txt to replay scripted prompts.
  • Run Streamlit interface: .venv/bin/python streamlit_app.py (opens UI to type questions, uses same LangGraph workflow).
  • Execute all automated checks: make test (invokes python run_tests.py) and make qa-gate for QA thresholds.
  • Generate observability evidence: python tests/generate_observability_evidence.py to create outputs/observability_*.json and markdown.
